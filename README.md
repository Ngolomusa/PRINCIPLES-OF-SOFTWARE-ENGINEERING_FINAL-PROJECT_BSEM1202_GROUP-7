# EDU CORE — Innovation and Learning

An advanced, interactive educational mobile application design that connects **Students**, **Teachers**, and **Parents** into a single, cohesive, AI-augmented ecosystem. Designed in **Figma**, **EDU CORE** aims to bridge the gap between traditional learning and contemporary technological needs, featuring gamified metrics, automated analytics, parent-teacher collaboration dashboards, and integrated AI tutoring.

---

##  Key System Features

The platform provides discrete, personalized interfaces depending on the target role mapped via an onboarding flow:

### 1. Student Ecosystem
* **Multi-Level Academic Support:** Categorized structural pipelines for *Secondary* and *Tertiary* education levels.
* **Modular Learning System:** Interactive course tiles spanning Chemistry, Mathematics, Geography, Physics, Computer Science, and Agricultural Science.
* **Dynamic Module Panel & Content Delivery:** In-app rich textual insights (e.g., Periodic Table breakdowns) coupled with dedicated media/document downloading nodes (`Video`, `Image`, `PDF`, `Audio`).
* **Gamified Testing Engine:** Integrated quiz mechanics with real-time score tracking, trophy iconography, and continuous flashcard evaluation loops.
* **Peer Engagement Hub ("Study Buddy"):** A matchmaking directory that pairs students based on overlapping study domains or performance statistics, equipped with native chat engines.

### 2. Teacher Dashboard
* **Class Overviews & Metrics:** Dedicated profiles displaying aggregated performance graphs across active student batches.
* **Curriculum Distribution & Monitoring:** Assignment queues, grade point averages, and course progression vectors.

### 3. Parent Dashboard
* **Real-time Progress Radar:** Visualization of continuous performance metrics including overall progress percentages (via custom pie charts), recent quiz scores, and course completion states.
* **Event & Task Notifications:** Automated alerts detailing submission milestones, upcoming exams, and academic breakthroughs.

### 4. Core Infrastructure & AI Layer
* **Omnipresent Navigation Bar:** Fast context switching across `Home`, `Course Directory`, `Profile Hub`, and `Global Search`.
* **AI Analytics Dashboard:** Automated insight generator featuring automated chat assistants, predictive grading charts, and continuous feedback metrics.

  ## 🛠️ Technology Stack & Tools Used

* **Design Platform:** Figma (Desktop Edition)
* **UI/UX Framework:** iOS Human Interface Guidelines (Target device: *iPhone 17 layout matrix*)
* **Design Paradigm:** Dark-mode primary structure accented with highly distinctive neon branding identifiers (`#00FF00` Electric Lime, deep royal blues, and high-contrast instructional warning tones for data blocks).
* **Prototyping Engine:** Figma Advanced Prototyping (utilizing smart-animates, overlay micro-interactions, drag-triggers, and hover states).

  ## Screen Breakdown Matrix

1.  **Welcome / Onboarding Screens:** Minimalist, atmospheric hero imagery featuring the green core logo alongside direct `"GET STARTED"` interaction pathways.
2.  **Authentication Matrix:** High-contrast, accessibility-compliant input panels supporting secure masked credentials, social identity providers (Google, TikTok, Facebook, Instagram), and contextual password restoration pathways.
3.  **Role Selector Screen:** Direct card selectors branching routing flows into specific behavioral spaces (`Student Portal`, `Teacher Portal`, `Parent Hub`).
4.  **Student Command Center:** Houses course selectors, active level identifiers, interactive navigation overlays, and real-time dashboard modules.
5.  **Interactive Content & Evaluation Panels:** Split-view frameworks handling markdown-like textbook reading, resource downloading panels, and instant-validation academic quizzes.
6.  **Progress Tracking & Collaborative Layers:** Data-dense layout structures compiling trend graphs, custom pie chart components, peer messaging grids, and multi-tier notification centers.

##  Project Setup Guide

To view or contribute to the interactive Figma prototype:

1.  Open your **Figma Desktop App** or visit [Figma Web](https://figma.com).
2.  Import the source design file or connect through the workspace file ID visible in your project configuration parameters.
3.  Switch workspace perspective from **Design** to **Prototype** mode in the right utility inspector.
4.  Click the **Play / Present** icon in the upper right action bar to engage with the live interactive prototype flow.
